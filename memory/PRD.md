# PRD — Волшебный Сад Знаний (Magic Garden of Knowledge)

## Original Problem Statement
Пользователь загрузил готовый ZIP-архив DIPLOMA.zip с полноценным веб-приложением и попросил развернуть его на сервере и предоставить ссылку.

## Architecture
- **Frontend**: React 19 + Tailwind CSS + Radix UI + Recharts
- **Backend**: FastAPI (Python) + Motor (async MongoDB driver)
- **Database**: MongoDB
- **Auth**: JWT + bcrypt для авторизации психологов

## User Personas
1. **Ребёнок** — проходит диагностические игры
2. **Психолог** — просматривает результаты диагностики через дашборд

## Core Requirements
- Лендинг с выбором роли (Ребёнок / Психолог)
- 8 диагностических игр: Фигуры, Цвета, Пазлы, Матрёшка, Найди лишнее, Найди пару, Память, Нелепицы
- Регистрация ребёнка (ФИО)
- Сохранение результатов игр
- Авторизация психолога (логин/пароль)
- Дашборд с результатами, статистикой и фильтрацией

## What's Been Implemented
- [2025-01-26] Полное развёртывание приложения из ZIP-архива
- Все API эндпоинты работают (100% тестов)
- Все фронтенд-маршруты работают
- Полный E2E-поток: регистрация ребёнка → игры → результаты → дашборд психолога
