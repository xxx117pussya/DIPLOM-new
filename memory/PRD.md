# PRD — Волшебный Сад Знаний (Magic Garden of Knowledge)

## Original Problem Statement
1. Пользователь загрузил готовый ZIP-архив DIPLOMA.zip с полноценным веб-приложением и попросил развернуть его на сервере
2. Пользователь загрузил шаблон ВКР и попросил создать дипломную документацию по проекту

## Architecture
- **Frontend**: React 19 + Tailwind CSS + Radix UI + Recharts
- **Backend**: FastAPI (Python) + Motor (async MongoDB driver)
- **Database**: MongoDB
- **Auth**: JWT + bcrypt

## User Personas
1. **Ребёнок** — проходит диагностические игры
2. **Психолог** — просматривает результаты диагностики через дашборд

## Core Requirements
- 8 диагностических игр с голосовым сопровождением
- Регистрация ребёнка (ФИО), авторизация психолога (JWT)
- Дашборд с результатами, статистикой, фильтрацией и экспортом в Excel

## What's Been Implemented
